# Inbox

Сюда попадают все новые заметки, идеи и сырые фрагменты.

## SLA обработки
- Разобрать и маршрутизировать в течение 24-72 часов.
- Если заметка неактуальна, удалять сразу, не хранить шум.

## Импорт из legacy Inbox
- Конвертация docx в pdf.
- docx -> html с помощью OpenXmlTools.
- html -> pdf с помощью DinkToPdf с предварительной установкой стилей в сгенерированный html.
- Эта версия не кроссплатформенная. С linux были проблемы.
- html -> pdf с помощью Haukcode.WkHtmlToPdfDotNet.
- HtmlToOpenXml - для парсинга html в docx (бесплатная и кроссплатформенная).
- LibreOffice - кроссплатформенный бесплатный конвертер документов в PDF.
