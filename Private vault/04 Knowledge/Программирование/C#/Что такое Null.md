# Что такое Null.

[[Основы платформы и языка|К разделу]] | [[Ссылочные и значимые типы. Стек и куча. Reference and value types (Ссылочные и значимые типы)|← Предыдущая]] | [[Оператор объединения с Null|Следующая →]]


Если объект относится к ссылочному типу, т.е. класс, то значением по умолчанию является null.

Создадим массив и не присвоим ему никакое значение:

![[Private vault/04 Knowledge/Documents/a08bd05439be99643b3a75ef80af5aee_MD5.png]]

В стеке выделилось место под ссылку на некие данные, которых ещё нет (мы не объявили). Null представляет собой пустую ссылку, т.е. когда есть переменная, которая должна хранить ссылку на данные, а данных нет.

Теперь присвоим значение массиву:

![[Private vault/04 Knowledge/Documents/bf9c08318f1394affcf52927047ce3fd_MD5.png]]

Присвоим массиву значение «null»:

![[Private vault/04 Knowledge/Documents/51addb27b58616f63c83b648e68e32a7_MD5.png]]

Таким образом связь между ссылкой и данными, указанная стрелкой разрывается.
