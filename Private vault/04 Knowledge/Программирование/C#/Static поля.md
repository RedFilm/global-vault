# Static поля
[[Ключевые слова и модификаторы|К разделу]] | [[Static|← Предыдущая]] | [[Static методы|Следующая →]]


Основной отличительной особенностью полей с данным модификатором является хранение в памяти. Если мы создаём статическое поле, то это поле будет доступно на уровне самого класса и будет общим для всех его объектов.
