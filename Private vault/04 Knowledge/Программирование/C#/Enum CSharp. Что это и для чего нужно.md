**Перечисления/enum C#** - это набор именованных констант, позволяющих сделать исходный код программы более удобочитаемым. Оно позволяет задать имена для небольшого набора используемых целочисленных значений.

![[Private vault/04 Knowledge/Documents/d4a50a077c6d625253be461f0440f7e4_MD5.png]]

Указание конкретных значений для каждой константы не обязательно, в таком случае нумерация будет начинаться с 0. При желании можно присвоить нужные значения для констант:

![[Private vault/04 Knowledge/Documents/3f3bb9f7ff1bf56a1581052ffe644416_MD5.png]]

Дефолтным типом для Enum считается тип Int, но при необходимости его можно изменить. Почему может возникнуть такая необходимость? Например, мы знаем, что Enum хранит в себе строго ограниченную область чисел для констант (0…40), тогда можно поменять тип на Byte, тем самым сэкономив место в оперативной памяти, и оптимизировав программу. Таким образом, если вы знаете, что ваша программа будет выполняться на машине, для которой критично кол-во оперативной памяти, можно выполнять подобные оптимизации для Enum. Синтаксис показан ниже:

![[Private vault/04 Knowledge/Documents/8fc4d5aaa3842478cd45aecf59380d1c_MD5.png]]
