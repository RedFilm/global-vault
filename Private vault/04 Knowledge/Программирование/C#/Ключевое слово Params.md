# Ключевое слово Params.
[[Ключевые слова и модификаторы|К разделу]] | [[Ключевое слово In|← Предыдущая]] | [[Static|Следующая →]]


Используется, когда есть необходимость перегрузки методов. Позволяет передавать различное кол-во параметров в один метод, без его перегрузок. Синтаксис представлен ниже:

![[Private vault/04 Knowledge/Documents/1193974b0f8a94deb36ce570fa4459c8_MD5.png]]

![[Private vault/04 Knowledge/Documents/ea69332d4eb2113118b2dda87876e577_MD5.png]]

В данном случае Params предоставляет возможность передачи параметров через запятую, в самом же методе идёт работа с массивом из этих параметров.

Если есть необходимость добавить дополнительные параметры, не входящие в массив, они объявляются перед Params:
