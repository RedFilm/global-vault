Существуют различные реализации CLI:

.NET Framework - (Только для Windows)

Micro Framework - (Для Embedded устройств)

Xamarin (Mono) - (Для Android, IOS, Windows)

.NET Core - (Кроссплатформенная)

.NET Core начиная с 5-ой версии называется .NET 5 без «Core».
