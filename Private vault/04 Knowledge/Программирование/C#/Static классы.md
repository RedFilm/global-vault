# Static классы
[[Ключевые слова и модификаторы|К разделу]] | [[Static конструкторы|← Предыдущая]] | [[Const Readonly|Следующая →]]


Статические классы могут содержать только статические члены.

Такие классы используются как правило для каких-то утилитарных вещей. Они призваны группировать в себе различные методы схожей тематики, которые обладают простым функционалом.

Стоит отметить, что злоупотреблять такими классами не стоит, т.к. такие классы не могут применяться при наследовании и реализации интерфейсов, соответственно при использовании статических классов мы не можем реализовать полиморфизм. У них есть проблемы с Dependency Injection и с тестированием.
