# Полезности
[[Полезности и рекомендации|К разделу]] | [[IAsyncResult|← Предыдущая]]


В проекте в строке «Расширения» (Visual Studio 2022) можно добавлять полезные надстройки.

Add New File - Shift+f2 добавляет файл в выбранную директорию.

Open Command Line (ALT + Space) - возможность работать с командной строкой.

ILSpy - позволяет просматривать ил-код. Иногда бывает полезно, например, можно отслеживать операции boxing и unboxing.
