# Static конструкторы
[[Ключевые слова и модификаторы|К разделу]] | [[Static методы|← Предыдущая]] | [[Static классы|Следующая →]]


\*\*

Статический конструктор работает только со статическими членами класса. В любом классе может быть только один статический конструктор, а также он не может принимать параметров.

В работе такого конструктора есть некоторые особенности, а именно:

- Статический конструктор вызывается только один раз, при этом не имеет значения сколько объектов класса мы создали
- Не обязательно создать экземпляр класса для вызова статического конструктора, он вызывается при любой попытке взаимодействия с классов, например вызова у класса статического метода
- Статический конструктор всегда вызывается в первую очередь, в момент обращения к классу
