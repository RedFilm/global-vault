# CRUD

[[Архитектурные подходы и паттерны|К разделу]] | [[Рекомендации|← Предыдущая]] | [[Паттерн посредник (Mediator)|Следующая →]]


CRUD - широко распространенный термин, который означает четыре стандартные операции над любой сущностью. CRUD - это акроним, который обозначает четыре операции над сущностями веб-разработки по первым буквам их названий: Create - создание Read - чтение Update - обновление Delete - удаление.
