# Const | Readonly
[[Ключевые слова и модификаторы|К разделу]] | [[Static классы|← Предыдущая]] | [[Преобразование и приведение типов в CSharp|Следующая →]]


Модификаторы const и readonly схожи, оба позволяют создавать переменные, значения которых нельзя изменить.

Разница заключается в том, что при создании поля с модификатором const мы должны сразу же поместить в него какие-то данные, иначе проект даже не скомпилируется, readonly допускает создание пустого поля, которое можно заполнить в момент выполнения программы через конструктор. Также важным отличием является модификатор const по умолчанию является статическим, хотя это и не объявляется явно, readonly же не является статическим.
