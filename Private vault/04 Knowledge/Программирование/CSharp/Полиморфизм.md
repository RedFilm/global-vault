# Полиморфизм
[[ООП и проектирование|К разделу]] | [[Наследование|← Предыдущая]] | [[Dependency injection|Следующая →]]


Полиморфизм - это способность функции работать с разными типами данных.

Существует 2 типа полиморфизма:

- **Add-hoc полиморфизм** (К нему относится приведение типов, перегрузка методов)
- **Параметрический полиморфизм** (Если есть функция, принимающая объект класса, у которого есть наследник, то такая функция может принимать объекты классов-наследников)
