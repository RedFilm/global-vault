# Область видимости. Контекст переменной.

[[Основы платформы и языка|К разделу]] | [[Принципы SOLID|← Предыдущая]] | [[Ссылочные и значимые типы. Стек и куча. Reference and value types (Ссылочные и значимые типы)|Следующая →]]


Контекст переменной - это фигурные скобки { }. Все что было создано в определённом контексте существует только внутри этого контекста. Если существует несколько переменных с одинаковым названием, но в разных контекстах, то внутри контекста будет использоваться переменная, созданная в той же области видимости.
