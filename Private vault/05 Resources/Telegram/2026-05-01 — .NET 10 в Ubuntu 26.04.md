# 2026-05-01 — .NET 10 в Ubuntu 26.04

## Кратко
- Ubuntu 26.04 LTS поддерживает `dotnet-sdk-10.0` из стандартного репозитория.
- Для .NET 8/9 предлагается PPA `dotnet/backports`.
- Подчеркнуты сценарии container и Native AOT.
